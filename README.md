
# LyricSync - 1.0.1 穩定封存版

這是一個專為 Android APK 與 PWA 體驗優化的 AI 卡拉 OK 播放器。

## 核心功能與更新 (1.0.1)
- **智慧捲動歌單**：播放時下方歌單會自動捲動，將當前歌曲保持在視覺中間位置。
- **播歌視覺優化**：正在播放的歌曲名稱改為居中顯示，且播放控制按鈕移至底部獨立區塊，不遮擋歌詞。
- **雙擊手勢**：在歌詞顯示區域「雙擊」即可切換播放/暫停。
- **介面個人化記憶**：系統自動儲存「字體大小」、「歌詞顏色」、「背景主題」。
- **AI 聽寫與同步**：上傳時若無歌詞，AI 自動聽寫產生同步 LRC。
- **原生環境優化**：打包 APK 後啟動自動隱藏偵測畫面，直接進入主介面。

## 如何將 APK 分享給他人？
1. **產出 APK**：
   - 執行 `npm run cap:open:android` 開啟 Android Studio。
   - 選擇 **Build > Build APK(s)**。
2. **分享檔案**：
   - 找到產出的 `app-debug.apk` 並分享給朋友。
3. **安裝提示**：
   - 提示對方需「允許安裝來自未知來源的應用程式」。
