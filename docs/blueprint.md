# **App Name**: LyricSync

## Core Features:

- MP3 Upload and Management: Allow users to upload and manage their MP3 music files.
- Playback Controls: Provide standard playback controls: play, pause, next, previous, and a draggable progress bar.
- Volume Control: Implement volume control functionality.
- Playlist Display: Display the current playlist with song titles and allow song selection.
- LRC File Generation: When a .txt lyric file is uploaded alongside the MP3, use AI tool to extract lyrics, timestamps and creates synchronized LRC files within the app for karaoke-style playback.
- Lyrics Display: If lyrics exist (either uploaded or generated via AI), display synchronized lyrics during playback, creating a karaoke (KTV) experience. Allow user to optionally provide their own text file
- AI-Powered Lyric Syncing: Tool for precise lyric-to-music alignment using AI analysis to ensure the lyrics displayed in real time match with song's progression.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a sophisticated feel.
- Background color: Light Gray (#F0F2F5) provides a clean and modern backdrop.
- Accent color: Teal (#009688) highlights interactive elements.
- Body text: 'PT Sans', a humanist sans-serif for clear and friendly readability.
- Headlines: 'Space Grotesk', a sans-serif, offers a futuristic, digital feel for primary app titles. Pair with 'PT Sans' for main text if needed.
- Use modern, minimalist icons for playback controls and other functions.
- Implement smooth transitions and subtle animations for a fluid user experience.
